# Shreem Consulting

Basic website for **Shreem Consulting** — Your Financial Friend.

**Live site:** [https://theshreemconsulting.github.io](https://theshreemconsulting.github.io)

## Local preview

Open `index.html` in your browser, or run a simple local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deployment

This repo is configured as a GitHub Pages user site. Push to the `main` branch and the site deploys automatically to `theshreemconsulting.github.io`.
